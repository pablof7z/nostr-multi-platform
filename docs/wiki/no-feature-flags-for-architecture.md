# No Feature Flags for Architectural Decisions — Name the Tradeoff and Pick a Side

## Architectural Decisions

Feature flags must not be used to defer architectural decisions. Tradeoffs must be named and a side chosen. There must be zero exceptions to clean architecture — all exceptions must be eliminated completely by finishing the work, not by feature-gating or removing incomplete features.
