# E2E Verification Driven by Agents Using REPL, CLI, or Xcode MCP

## E2E Verification Approach

E2E verification should be driven by agents using REPL, CLI, or Xcode MCP rather than manual testing. When validating UI changes, run the binary locally and visually inspect the output rather than relying solely on cargo test.
