# Activity Tab and Navigation Structure

## Activity Access & Navigation

The Activity item is accessed via a top-right toolbar bell icon rather than a bottom tab bar item. Tapping the bell toolbar button presents NotificationsView in a NavigationStack sheet. [^cb337-1]


## Bottom Tab Bar

The bottom tab bar contains five tabs rather than six. Reducing the tab count from six to five eliminates the duplicate back button on the Accounts settings screen. [^cb337-2]

DMs and groups are separated into distinct tabs: a 'Chats' tab for DMs and a 'Groups' tab for groups, rather than mixing them together. [^eb342-1]

The tab bar order is: Home (house.fill), Chats (bubble.left.and.bubble.right.fill), Groups (person.3.fill), Wallet (bolt.fill), Settings (gearshape.fill). [^eb342-2]
