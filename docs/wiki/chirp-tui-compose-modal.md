# Chirp TUI — Compose Modal

## Compose Modal Overlay

Composing a new tweet or reply in chirp-tui opens a centered ratatui modal overlay. The modal uses a Clear-backed overlay with an ACCENT_CYAN border and a block cursor character. It displays a character count. [^1ca92-1]


## Keybindings

Pressing Enter sends the composed tweet or reply. Pressing Shift+Enter inserts a newline in the compose modal. [^1ca92-2]
