## Limitations

- This is a deterministic prototype harness. Real relay sockets, LMDB/SQLite writes, UniFFI marshaling, platform wrappers, and NSE calls are modeled because those runtime pieces do not exist yet.
- Replay mode is CI-shaped and deterministic. Capture mode currently writes a synthetic trace manifest, not live WebSocket frames.
- Live mode is intentionally not implemented until the actor, relay adapter, and storage backend exist.

## Observations
